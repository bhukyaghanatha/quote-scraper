# Quote Scraper

A Python web scraping project that extracts quotes and authors from quotes.toscrape.com and saves them to a timestamped file.

## Features
- HTTP request handling with error management
- HTML parsing using BeautifulSoup
- Structured data extraction
- Automatic output folder creation
- Timestamped output file

## Technologies Used
- Python
- Requests
- BeautifulSoup

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the script:
   python scraper.py
